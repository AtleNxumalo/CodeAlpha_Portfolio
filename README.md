# Personal Portfolio Website 
A responsive portfolio web application built using HTML, CSS, and JavaScript to showcase my projects, skills, and design work.

## Features

* Dynamic project gallery rendered using JavaScript
* Category-based filtering (web, app, UI/UX, graphics)
* Interactive project cards with hover effects
* Modal view with project details and media carousel
* Support for both images and videos
* Mobile-friendly design with responsive layout and touch support

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* Font Awesome
* Google Fonts (Inter)

## Project Structure

```
portfolio/
├── index.html
├── stylesheet.css
├── scripts.js
├── assets/
└── README.md
```

## Key Functionality

* Projects are stored in a JavaScript array and rendered dynamically
* Filter system updates displayed projects based on category
* Modal displays project details with a carousel for media
* Carousel supports navigation buttons and swipe gestures on mobile

## Projects Included

This portfolio includes a range of work such as:

* Web applications
* Mobile applications
* UI/UX prototypes
* Graphic design projects
